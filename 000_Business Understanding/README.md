# سناریوی کسب‌وکار

یک مرکز تخصصی قلب و عروق با افزایش تعداد بیماران مواجه شده است.

انجام آزمایش‌های تخصصی قلب برای تمامی بیماران:

* زمان‌بر است
* هزینه‌بر است
* نیازمند تجهیزات محدود بیمارستان است

مدیریت بیمارستان قصد دارد قبل از انجام آزمایش‌های تخصصی، بیماران پرریسک را شناسایی کند.

هدف آن‌ها توسعه یک سیستم هوشمند است که بتواند احتمال ابتلا به بیماری قلبی را پیش‌بینی کند.

شما به عنوان مشاور داده‌کاوی برای بررسی امکان‌سنجی پروژه استخدام شده‌اید.
