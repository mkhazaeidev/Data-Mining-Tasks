# مقدمه

در مرحله Modeling شما:

* نوع مسئله را شناسایی کردید.
* الگوریتم‌های مختلف را آزمایش کردید.
* مدل‌های مختلف ساختید.
* مدل‌ها را ارزیابی کردید.
* بهترین مدل را انتخاب کردید.

اما هنوز یک سؤال مهم باقی مانده است:

> آیا این مدل واقعاً برای استفاده در بیمارستان مناسب است؟

هدف مرحله Evaluation پاسخ دادن به همین سؤال است.
