# مخزن تسک‌های دوره داده‌کاوی (Data Mining Assignments)

به مخزن رسمی تسک‌های دوره آموزش داده‌کاوی خوش آمدید.

در این ریپازیتوری، تمامی تمرین‌ها، پروژه‌ها و تسک‌های هفتگی دوره منتشر می‌شوند. دانشجویان موظف هستند در بازه‌های زمانی مشخص، تسک‌های مربوط به هر جلسه را دانلود کرده و پس از انجام آن‌ها، خروجی مورد نیاز را تحویل دهند.

---

## هدف این ریپازیتوری

این مخزن با اهداف زیر ایجاد شده است:

* انتشار تسک‌های هفتگی دوره
* ارائه دیتاست‌های مورد نیاز تمرین‌ها
* ارائه توضیحات و راهنمای انجام تسک‌ها
* ایجاد یک مرجع متمرکز برای تمامی تمرین‌های دوره
* کمک به دانشجویان برای یادگیری عملی مفاهیم داده‌کاوی

---

## ساختار فایل‌ها

هر تسک با یک شماره مشخص در مخزن قرار می‌گیرد.

نمونه:

```text
task001_data_understanding.ipynb
task002_data_preparation.ipynb
task003_feature_engineering.ipynb
...
```

شماره تسک‌ها به ترتیب جلسات و سرفصل‌های دوره افزایش پیدا می‌کند.

---

## نحوه دانلود تسک‌ها

### روش اول: دانلود کل ریپازیتوری

1. وارد صفحه اصلی ریپازیتوری شوید.
2. روی دکمه سبز رنگ **Code** کلیک کنید.
3. گزینه **Download ZIP** را انتخاب کنید.
4. فایل ZIP را دانلود و استخراج (Extract) کنید.
5. فایل تسک مورد نظر را باز کنید.

---

### روش دوم: استفاده از Git (پیشنهادی)

اگر Git روی سیستم شما نصب است:

```bash
git clone REPOSITORY_URL
```

پس از دانلود:

```bash
cd repository-name
```

برای دریافت آخرین تسک‌ها:

```bash
git pull
```

## نحوه انجام تسک‌ها

1. فایل Notebook مربوط به تسک را باز کنید.
2. توضیحات هر بخش را به دقت مطالعه کنید.
3. کدهای مورد نیاز را تکمیل کنید.
4. خروجی‌ها را بررسی و تحلیل کنید.
5. Notebook تکمیل‌شده را ذخیره کنید.
6. طبق دستور مدرس، فایل نهایی را ارسال نمایید.

---

## نکات مهم

* صرفاً اجرای کد کافی نیست.
* برای تمامی تحلیل‌ها باید توضیح و نتیجه‌گیری ارائه شود.
* از کپی کردن تحلیل سایر دانشجویان خودداری کنید.
* تمرکز اصلی دوره بر درک داده، تحلیل مسئله و تفکر داده‌محور است.
